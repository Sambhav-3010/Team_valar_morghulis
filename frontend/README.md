# Engineering Intelligence Dashboard

The frontend application for visualizing engineering metrics, team velocity, and AI-driven insights. Built with Next.js and Tailwind CSS.

## Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **State Management**: React Context / Hooks

## Features
- **Dashboard**: High-level overview of DORA metrics and team health.
- **Projects View**: Detailed breakdown of activities per project.
- **Team View**: Work-life balance and collaboration insights.
- **AI Insights**: Natural language summaries of project blockers and risks.

## Setup
### Environment Variables
Create a `.env.local` file:
```bash
NEXT_PUBLIC_API_URL=http://localhost:3001/api
```

### Running
```bash
# Install dependencies
npm install

# Start development server
npm run dev
# App runs at http://localhost:3000
```
